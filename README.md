# 習慣トラッカー

毎日の習慣を記録するシンプルなWebアプリ。iPhoneのホーム画面に追加して使えます。

## 機能

- 習慣のチェック（タップで完了／取り消し）
- 「編集」ボタンで編集モードに切り替え、習慣名やアイコンを変更（誤編集防止）
- 過去30日まで遡って入力可能（日付の◀▶、または「この1週間」のマスをタップ）
- データの書き出し／読み込み（JSON）でバックアップや端末間移行が可能
- 連続記録日数（全体／習慣ごと）
- 退職日までのカウントダウン（カードをタップして設定）
- 習慣の追加・編集・削除
- 直近1週間の履歴
- 過去4ヶ月のヒートマップ・最長連続日数
- ライト／ダークモード自動切り替え

## 使い方

`index.html` をブラウザで開くだけで動きます。データはブラウザのローカルストレージに保存され、外部には送信されません。

## iPhoneで使う

1. このリポジトリの GitHub Pages を有効化（Settings → Pages → Branch: `main` / `/ (root)`）
2. 公開された URL を iPhone の Safari で開く
3. 共有ボタン →「ホーム画面に追加」

## 構成

| ファイル | 役割 |
|---|---|
| `index.html` | 画面構造 |
| `style.css`  | スタイル |
| `app.js`     | ロジック・状態管理 |
| `manifest.webmanifest` | PWA 設定 |
| `icon.svg`   | アプリアイコン |

ビルド不要・依存なしの素のHTML/CSS/JSです。
